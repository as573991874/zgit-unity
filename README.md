# zgit-unity
用于管理 unity 依赖库的方案

## 工具具有以下命令待开发

```
--version
	查看工具版本

--status
	查看工作空间执行状态

--init_workspace
	初始化空间项目配置

--init_repo_config
	初始化拉取配置仓库

--pull_config
	拉取 / 更新配置仓库

--push_config
	推送 / 应用配置仓库更新

--create_config_setting
	创建一份配置

--create_config_setting_copy
	创建一份复制的配置

--open_config
	打开配置仓库

--open_repo
	根据传入的参数打开仓库

--ls_config
	输出可用配置

--ls_project
	输出已拉取的工程

--init_project
	初始化项目，指定目录与配置

--switch_project
	切换项目目录

--pull
	批量拉取分支，是否强制拉取

--push
	批量推送分支

--checkout
	批量切换分支

--branch
	批量创建分支
```
